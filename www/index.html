mail-assistant

IMAP → LLM → Google Tasks — automatic email triage for your inbox

Home Privacy Policy GitHub

What it does

  • Connects to your IMAP mailbox and fetches new emails since a configurable start date
  • Sends each email to an LLM (Ollama, Anthropic Claude, or Mistral) for classification
  • Creates a Google Task with priority and due date when an email requires action
  • Moves emails to suggested IMAP folders
  • Automatically marks tasks as done when the email is replied to or archived

What it never does

  • never Deletes emails — no EXPUNGE or DELETE
  • never Marks emails as read
  • never Replies to or sends emails
  • never Sends email content to third parties when using Ollama

Google API access

mail-assistant uses Google OAuth 2.0 to access Google Tasks on your behalf. Only the following scope is requested:


ScopePurpose
https://www.googleapis.com/auth/tasks Read, create and update tasks in Google Tasks

No access to Gmail, Google Drive, Calendar or any other Google service is requested.
